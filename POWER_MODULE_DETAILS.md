# ⚡ รายละเอียดโมดูลจ่ายไฟ - Breadboard Power Supply Module

## 📋 ภาพรวมโมดูลจ่ายไฟ

จากรูปที่ได้รับ เป็น **Breadboard Power Supply Module** ที่มีคุณสมบัติดังนี้:

## 🔌 ข้อมูลจำเพาะทางเทคนิค

### Input (อินพุต)
- **แรงดันเข้า**: 7-12V DC
- **กระแสสูงสุด**: 2-3A (ขึ้นอยู่กับรุ่น)
- **ตัวเชื่อมต่อ**: DC Jack หรือ Terminal Block
- **แหล่งจ่าย**: Jetson Nano Pin 2 (5V) + Pin 6 (GND)

### Output (เอาต์พุต)
- **5V Rail**: 5.0V DC ±5%
- **3.3V Rail**: 3.3V DC ±5%
- **กระแสสูงสุด 5V**: 1-2A
- **กระแสสูงสุด 3.3V**: 800mA-1A
- **ขั้วเอาต์พุต**: Pin Headers ทั้งสองข้าง

### คุณสมบัติเพิ่มเติม
- **LED สถานะ**: สีเขียว (แสดงการทำงาน)
- **USB Port**: สำหรับจ่ายไฟ 5V (ถ้ามี)
- **สวิตช์ ON/OFF**: สำหรับแต่ละ Rail
- **ป้องกันไฟเกิน**: Built-in Protection

## 🔧 การเชื่อมต่อกับระบบ

### การจ่ายไฟเข้า (Power Input)
```
Jetson Nano → Power Module
├── Pin 2 (5V) → DC Input Positive (+)
├── Pin 6 (GND) → DC Input Negative (-)
└── ตรวจสอบ LED สีเขียวติด
```

### การจ่ายไฟออก (Power Output)
```
Power Module → Components
├── 5V Rail (Left/Right) → ESP32 VIN, PM2.5 Sensor
├── 3.3V Rail (Left/Right) → DHT22, MQ-135
├── GND Rail (Left/Right) → All Ground Connections
└── USB Port (ถ้ามี) → Additional 5V Devices
```

## 📊 การใช้ไฟฟ้าของระบบ

### การคำนวณกำลังไฟฟ้า

| Rail | อุปกรณ์ที่เชื่อมต่อ | กระแส (mA) | กำลัง (W) |
|------|-------------------|------------|-----------|
| **5V** | ESP32 | 80-250 | 0.4-1.25 |
| **5V** | PM2.5 Sensor | 70-300 | 0.35-1.5 |
| **3.3V** | DHT22 | 1.5-2.5 | 0.005-0.008 |
| **3.3V** | MQ-135 | 150-200 | 0.5-0.66 |
| **รวม 5V** | - | 150-550 mA | 0.75-2.75 W |
| **รวม 3.3V** | - | 151.5-202.5 mA | 0.505-0.668 W |

### ความปลอดภัย
- **5V Rail**: ใช้ ~550mA จากความสามารถ 1-2A ✅ ปลอดภัย
- **3.3V Rail**: ใช้ ~200mA จากความสามารถ 800mA-1A ✅ ปลอดภัย

## 🛠️ การติดตั้งและใช้งาน

### ขั้นตอนการติดตั้ง

1. **เตรียมโมดูล**
   ```
   - ตรวจสอบโมดูลไม่เสียหาย
   - ตรวจสอบ Pin Headers ไม่งอ
   - ทำความสะอาดขั้วต่อ
   ```

2. **เชื่อมต่อไฟเข้า**
   ```
   - เชื่อมต่อ Jetson Pin 2 (5V) → โมดูล DC Input (+)
   - เชื่อมต่อ Jetson Pin 6 (GND) → โมดูล DC Input (-)
   - ตรวจสอบขั้วไฟถูกต้อง (+ กับ +, - กับ -)
   ```

3. **ทดสอบการทำงาน**
   ```
   - เปิดไฟ Jetson Nano
   - ตรวจสอบ LED สีเขียวติด
   - วัดแรงดัน 5V Rail = 4.8-5.2V
   - วัดแรงดัน 3.3V Rail = 3.2-3.4V
   ```

4. **เชื่อมต่ออุปกรณ์**
   ```
   - เชื่อมต่อสายไฟไปยังเซ็นเซอร์ต่างๆ
   - ใช้สีสายไฟตามมาตรฐาน
   - ตรวจสอบการเชื่อมต่อทั้งหมด
   ```

## 🔍 การแก้ปัญหา

### ปัญหาที่พบบ่อย

**1. LED ไม่ติด**
```
สาเหตุ:
- ไฟเข้าไม่ถูกต้อง
- ขั้วไฟสลับ
- โมดูลเสียหาย

วิธีแก้:
- ตรวจสอบแรงดันไฟเข้า
- ตรวจสอบขั้ว + และ -
- เปลี่ยนโมดูลใหม่
```

**2. แรงดันเอาต์พุตไม่ถูกต้อง**
```
สาเหตุ:
- โหลดมากเกินไป
- การเชื่อมต่อหลวม
- โมดูลร้อนเกินไป

วิธีแก้:
- ลดโหลดที่เชื่อมต่อ
- ตรวจสอบการเชื่อมต่อ
- เพิ่มการระบายความร้อน
```

**3. อุปกรณ์ทำงานไม่เสถียร**
```
สาเหตุ:
- แรงดันไฟตก
- สัญญาณรบกวน
- กราวด์ไม่ดี

วิธีแก้:
- เพิ่มตัวเก็บประจุ (Capacitor)
- ใช้สายไฟสั้นและหนา
- ตรวจสอบการต่อกราวด์
```
